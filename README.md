# Deep Darkness

![game-preview](/ReadmeStatic/preview.png)

> One day, a **brave knight** wandered into a dark dungeon, and the gates behind him suddenly closed, seemingly forever.
The only way out was through the **deep darkness**, teeming with the local inhabitants...

**Mode**: Singleplayer\
**Genre**: Coding, Roguelike, Puzzle\
**Platforms**: Windows 

## Description

The player will control the knight by entering commands into the **terminal**.

**Any constructions are possible**, as long as the knight ultimately reaches the end of the level.

The player can find descriptions of all the commands in the **Book of Knowledge**, but how and where to use them to complete the level is up to the player to decide...
